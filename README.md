## Naming Convention

Files follow the pattern: `{zero-padded-4-digit-id}_{snake_case_problem_name}.cpp`

Examples:
- `0001_two_sum.cpp`
- `0146_lru_cache.cpp`
- `0994_rotting_oranges.cpp`

## Workflow

1. Open problem via VS Code LeetCode extension
2. Implement solution
3. Test against sample cases (Test button)
4. Submit to LeetCode (Submit button)
5. Commit and push
